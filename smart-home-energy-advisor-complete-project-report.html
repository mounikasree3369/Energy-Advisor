1. Project Overview

The Smart Home Energy Advisor is a production-ready, AI-powered web application that connects household smart meter data to IBM's Granite large language models on watsonx.ai. The AI agent — named Aria — analyses energy consumption patterns, predicts electricity bills, identifies energy waste, and delivers personalised recommendations through an interactive chat interface and real-time analytics dashboard.

The application is designed to be genuinely useful for citizens — enabling homeowners, tenants, and energy-conscious households to understand and reduce their electricity costs using cutting-edge AI, without needing any technical knowledge.

Key Achievement: Full IBM Granite AI integration with live watsonx.ai inference, ₹ INR currency support, interactive electricity bill calculator from meter readings, 6 Chart.js visualisations, and a 4-tab responsive SPA — all on IBM Cloud Lite (free tier).

7. Electricity Bill Calculator (Meter Reading)

Located in the Analytics tab. Citizens enter their actual meter readings to get an accurate bill estimate — no smart meter required.

Input Fields

FieldDescriptionExample
Previous Reading (kWh)Meter reading at start of billing period4500
Current Reading (kWh)Meter reading at end of billing period5083
Rate per Unit (₹/kWh)Tariff rate from electricity bill₹8
Fixed/Meter Charge (₹)Monthly standing/service charge₹50

Calculation Formula

Units Consumed  = Current Reading − Previous Reading
                = 5083 − 4500 = 583 kWh

Energy Charge   = Units × Rate
                = 583 × ₹8 = ₹4,664

Tax (8%)        = ₹4,664 × 8% = ₹373.12

Total Bill      = Energy Charge + Fixed Charge + Tax
                = ₹4,664 + ₹50 + ₹373.12 = ₹5,087.12

Carbon Footprint = Units × 0.386 kg/kWh
                 = 583 × 0.386 = 225.04 kg CO₂

Annual Projection = Total Bill × 12 = ₹61,045.44/year

Output Provided

  • ✅ 4 stat cards: Units consumed, Total bill (₹), Carbon (kg CO₂), Daily average
  • ✅ Full bill breakdown table (energy charge, fixed charge, tax, total)
  • ✅ Usage category label (Very Low / Below Average / Average / High)
  • ✅ Annual projection
  • ✅ Personalised energy-saving tip based on usage level

10. Setup & Installation

Prerequisites

  • Python 3.11+ (tested on Python 3.14)
  • IBM Cloud account (free Lite tier)
  • IBM watsonx.ai project with Watson Machine Learning (WML) Lite service

Local Installation Steps

  1. Create and activate virtual environment: python -m venv venv then .\venv\Scripts\activate
  2. Install all dependencies: pip install -r requirements.txt
  3. Copy environment template: cp .env.example .env
  4. Fill in IBM_API_KEY and WATSONX_PROJECT_ID in .env
  5. Start the server: python app.py
  6. Open browser at: http://localhost:5000

12. IBM Cloud Lite Deployment

Option A — IBM Cloud Foundry

# 1. Install IBM Cloud CLI and login
ibmcloud login --sso
ibmcloud target --cf

# 2. Set credentials as environment variables (never in manifest.yml)
ibmcloud cf set-env smart-home-energy-advisor IBM_API_KEY "your_key"
ibmcloud cf set-env smart-home-energy-advisor WATSONX_PROJECT_ID "your_id"
ibmcloud cf set-env smart-home-energy-advisor FLASK_SECRET_KEY "random_string"

# 3. Deploy
ibmcloud cf push

# 4. Open app
ibmcloud cf open smart-home-energy-advisor

Option B — IBM Code Engine (Serverless)

ibmcloud ce application create \
  --name energy-advisor \
  --image us.icr.io/namespace/energy-advisor:latest \
  --port 5000 \
  --env IBM_API_KEY=your_key \
  --env WATSONX_PROJECT_ID=your_id

What to Ask the AI (Sample Questions)

  • "Which appliances use the most energy?"
  • "When is the cheapest time to run my washing machine?"
  • "How can I reduce my carbon footprint?"
  • "Should I install solar panels? Will it save money?"
  • "Give me your top 5 energy saving tips"
  • "What is my home energy efficiency score?"
  • "How much can I save by shifting my AC usage to night?"

Bill Calculator — Step by Step

  1. Go to Analytics tab
  2. Find your meter box at home — note the reading shown (e.g. 5083)
  3. Check last month's bill for the previous reading (e.g. 4500)
  4. Enter your ₹/unit rate from your electricity bill (typically ₹6–₹10/unit)
  5. Enter your fixed charge (monthly meter rental, usually ₹40–₹100)
  6. Click "Calculate My Electricity Bill"
  7. Review the breakdown: units consumed, energy charge, tax, total bill, CO₂
